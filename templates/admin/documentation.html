Admin Dashboard Overview

Welcome to the HR Management System's admin documentation. This guide will help you understand and utilize all administrative features available to manage your organization effectively.

Employee Management

Adding Employees

  • Navigate to "Add Employee" in the navigation menu
  • Fill in required employee details
  • Upload profile picture (optional)
  • Set initial password

Managing Employees

  • View all employees
  • Edit employee information
  • Manage departments
  • Update salary information

Credential Management

Access the Credentials page to manage user passwords and view login history.

Password Management

  1. Click on "Reset Password" for any user
  2. Enter new password in the modal
  3. Confirm to update credentials
  4. Notify user of password change

Login History

  • View all login attempts
  • Track successful/failed logins
  • Monitor suspicious activities
  • Export login history

Security Features

Always follow security best practices when managing user credentials.

Security Guidelines

  • Use strong passwords (minimum 8 characters)
  • Regular password rotation
  • Monitor login attempts
  • Secure credential storage
