HR Management System

A comprehensive HR management solution built with Flask and SQLite, designed for small to medium-sized organizations.

Features

  • Employee Management
  • Attendance Tracking
  • Leave Management
  • Task Assignment
  • Performance Monitoring
  • Document Management

Installation

git clone https://github.com/yourusername/hr-management.git
cd hr-management
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

Configuration

Configure your database and application settings:

flask init-db  # Initialize the database
# Update config.py with your settings

Usage

Start the development server:

flask run
